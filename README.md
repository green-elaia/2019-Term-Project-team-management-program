# 전공기초프로젝트 과목 Term Project: 대학생 팀프로젝트 관리 프로그램

본 프로젝트는 다음의 기능을 구현한 대학생 팀프로젝트 관리 프로그램 입니다.

- 메모파일 작성, 저장, 불러오기
- 랜덤 팀장 선출 및 팀 정보 관리
- 캘린더를 이용한 To Do list 관리
- 팀원들의 과목시간표 이미지로 공강 시간을 파악하여 모임 가능시간 찾기

개발 Tool: C++, QT, SQLite, OpenCV
